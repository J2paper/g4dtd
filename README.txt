설치방법
========
1. 압축을 풀고 FTP로 업로드 합니다.
2. 브라우저에서 실행합니다.
3. "루트 디렉토리에 data 디렉토리를 만들어 주시기 바랍니다." 라는 메세지가 나오면
4. $> mkdir data 명령으로 data 디렉토리를 만듭니다.
5. "data 디렉토리의 퍼미션을 707로 변경하여 주십시오." 그 다음 설치하여 주십시오." 라는 메세지가 나오면
6. $> chmod 707 data 명령으로 data 디렉토리의 퍼미션을 변경합니다.
7. 브라우저를 새로고침하여 설치합니다.
